Problem : Speedo kadang mati kadang nyala

Solution : Adit :

kalo kadang idup kadang mati kemungkinannya ada 3..&#x20;

1\. Masalah di seputaran kabel speed sensor.. Bisa socket kurang nyolok ato kabel ada indikasi mau putus2. Speedometer sensor udah saatnya diganti.. Untuk tes normal tidaknya speed sensor bisa dilakukan dgn cara cabut speedo sensornya (kabel socket dlm keadaan masih terpasang) lalu putar2 plastik kecil yg ada di ujung speedo sensor dgn posisi kontak on.. Kalo saat diputar2 jarum speedometer normal naik turunnya berarti speedo sensor tidak ada masalah..3. Speedometer gear pinion jebol.. Ini yang paling susah krn barangnya susah didapat n untuk mengganti gear ini gearbox harus dibongkar..

Sebagai info, yg sy pernah alami adalah kasus no 2 & 3.. Speedo sensor sy mints kirim langsung dr malaysia..

Efek umum yg ditimbulkan saat speedometer mati adalah mobil jerking & boros ngga ketulungan

AKI FOR SAVVY  :

Aki mobil yg di rekomendasi buat savvy adalah Type GS MF NS60LS (kering). Atau Aki Merk Rocket (lebih murah) dgn type yg sama dengan GS.

AKI yg mobil jarang di pakai sampai berhari-hari sebaiknya kabel negatifnya (-) dicopot, kalo mau di pake baru di pasang.

Cara Ngejumper'nya :

\- Pake kabel jumper yg serat kabelnya tebal biar tdk panas.

\- Waktu ngejumper, mobil yg mau ngejumper harus dalam keadaan hidup sambildi Gas di RPM tinggi > 2000 rpm

\- Kalo mobil yg Aki nya bagus mau ngejumper ke mobil yg Aki nya soak tp waktu ngejumper gak di hidupin, maka besar kemungkinan jadi ikutan soak.
